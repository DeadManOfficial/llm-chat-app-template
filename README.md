| **Mobile Ready** | Responsive design |

## Quick Start

```bash
git clone https://github.com/DeadManOfficial/llm-chat-app-template.git
cd llm-chat-app-template
npm install
npm run dev
```

Open `http://localhost:8787`

---

## Deploy

```bash
npm run deploy
```
